About Me

Afton Hencky is a Smith College alumna and "techie" living in the Bay Area. She is a technology enthusiast who previously lived and worked abroad as an web development intern for a non-profit study abroad program. Her computer skills include HTML, CSS, Python, Django, Git/SVN, JavaScript/jQuery, and WordPress. She is currently learning Ruby on Rails and is big fan of online education platforms such as Coursera and edX. She recently completed General Assembly's front-end web development course and was recently accepted into Dev Bootcamp in San Francisco. In addition to her technical interests, Afton has a B.A in Psychology, with strong research, writing, and editing skills.

Projects

TLC - the Teaching and Learning Center

A microsite built with the Django web framework during my internship with the Danish Institute for Study Abroad in Copenhagen. The site is used to host information, resources and newsletters for DIS faculty and academic staff. Content is managed by staff through Django's admin interface.

Student Blogger Page for DIS

A Django application designed to showcase students' experiences studying abroad in Copenhagen. RSS feeds are aggregated using the python feedburner library. The page updates with blog posts from students' individual blogs from Blogspot and other platforms.

Tuner - SoundCloud API project

This project was created during General Assembly's front-end web development course. It allows users to find music on SoundCloud using the SoundCloud API to filter songs based on search criteria and tags.
