About

Hello! I'm Afton, a Smith College alum and queer techie living in the Bay Area. I'm a technology enthusiast with experience in web development and user support. I've previously worked as an IT assistant with the Danish Institute for Study Abroad in Copenhagen and as a copy editor for Metamorphoses, a 5-college journal for literary translations. I hold a B.A in Psychology, with strong skills in research, writing, and editing.

I'm currently seeking professional opportunities in the Bay Area. Check out my resume and LinkedIn profile and feel free to contact me.

Projects

TLC - the Teaching and Learning Center

A microsite built with Django during my internship with the Danish Institute for Study Abroad in Copenhagen. The site is used to host information, resources and newsletters for DIS faculty and academic staff. Content is managed by staff through Django's admin interface.

Student Blogger Page for DIS

A Django app RSS feed aggregator using the python feedburner library. The page updates with blog posts from students' individual blogs.

Tuner - SoundCloud API project

My project during General Assembly's front-end web development course finds music on SoundCloud using their API to filter songs based on search criteria.
